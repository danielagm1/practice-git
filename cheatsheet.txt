GitHub commands:
1. git init -> initialize local repository as git repository (folder or directory)
  (prepare local repository to be attached to the one on GitHub

2. paste the line you copied from GitHub

3. git remote -v -> to check if your local repo is connected to github (you should see origin github.com/danielagm1/repository-name.git)

2. git add -> add . all files from the repository to the staging phase (files are still on my computer but well packaged and ready to be shipped)

3. git commit -m"message goes here" -> attach a message to the commit

4. git push origin master -> sends the files to the corresponding repository on GitHub
